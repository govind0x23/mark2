## Deployment

To deploy this project run


```
python manage.py runserver
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/govind0x23/mark2
```

Go to the project directory

```bash
  cd mark2
```



Start the server

```bash
  python manage.py runserver
```


## Tech Stack

**Client:** HTML,CSS,Bootstrap 

**Server:** Python, Django
